# Design System

Đặt file design system của bạn vào folder này.

## Hỗ trợ các format:
- `.md` — Markdown mô tả color palette, typography, spacing,...
- `.json` — Design tokens (colors, fonts, spacing, border-radius,...)
- `.txt` — Text mô tả style guidelines

## Nội dung nên có:

### Colors
- Primary, secondary, accent colors (hex hoặc RGB)
- Background colors
- Text colors
- Status colors (success, error, warning, info)

### Typography
- Font family (tên font, Google Fonts link nếu có)
- Font sizes (heading, body, caption,...)
- Font weights

### Spacing
- Base spacing unit (4px, 8px,...)
- Common spacing values

### Border Radius
- Button radius
- Card radius
- Input radius

### Component Styles (nếu có)
- Button styles (primary, secondary, ghost,...)
- Card styles
- Input styles

## Nếu không có design system
Skill sẽ dùng default style: neutral gray palette, system font, 8px spacing grid, rounded-xl corners.
